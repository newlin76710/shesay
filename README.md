# SheSay Vercel Site

Next.js + Tailwind landing page for SheSay.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Import this folder (`shesay-vercel`) into Vercel.
2. Framework preset: Next.js.
3. Build command: `npm run build`
4. Output: default Next.js output.

## Notes

- Replace `https://line.me/` in `app/page.tsx` and `components/cta-banner.tsx` with your real LINE or booking URL.
- Update metadata in `app/layout.tsx` if needed.
